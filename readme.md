# Delete Objects By Regex

## 使い方
ヒエラルキーのオブジェクトを右クリックで「Delete Objects By Regex」が追加されます。
正規表現にマッチする子孫オブジェクトをすべて削除します。
間違えたらUndoできます。

## インストール
DeleteObjectsByRegex.csを適当な場所に保存します。
https://yukiyukivirtual.booth.pm
にあるunitypackageを使ってもいいです。
